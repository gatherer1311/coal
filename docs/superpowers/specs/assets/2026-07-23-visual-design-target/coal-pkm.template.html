6# Minibuffer Design
5
4The minibuffer is the single point of command in [[Coal]] —
3every action funnels through one prompt at the bottom of the
2frame, like [[Emacs]] but for a networked vault.
1
13## Why bottom, full width
1
2- Predictable focus — eyes land in one place, always
3- Completions grow upward, never covering the modeline
4- Echo messages reuse the same line when idle
5
6## Interactions
7
8- [x] M-x opens command completion
9- [x] Fuzzy match on name + description
10- [ ] Inline argument prompts (find-note:)
11- [ ] History ring on M-p / M-n
12
13See also [[Command Palette vs Minibuffer]] #ux #keyboard
