# Coal

A minimal, Linux-native, keyboard-first text editor **whose value is its plugin API**. The kernel is a
fast, hackable, **general-purpose** editor with the *hands of Emacs* — a central minibuffer, deep
hackability, and an Emacs-modeled **command + keybinding system** (a curated default keymap, every
command rebindable) — that opens and edits **any** file format from the keyboard, like Emacs, Vim, or
VSCode, with zero plugins. A rich **opt-in bundled plugin suite** then adds the *head of Obsidian*: a
full Markdown/Org PKM workspace — plain-text notes, links, backlinks, and live preview.
